Note: Vendor press release (2026-01-26) — Agentic Identity Hub 2.0 features, deployment, target customers.

Key facts:
- What: dedicated identity infrastructure to manage AI agents and MCP servers alongside human users with enterprise controls.
- Announcement date: January 26, 2026.
- Features:
  - MCP auth: OAuth 2.1, PKCE, DCR (Dynamic Client Registration), CIMD support; protocol-compliant authorization for MCP servers.
  - AI agent identity: dedicated identities, tool-level scopes, attribute assignment.
  - Non-human identity: manages agents as "first-class identities" distinct from user/service-account models.
  - Delegated access: credential vault storing OAuth tokens + API keys, 50+ prebuilt connection templates for third-party access.
  - SIEM export: stream audit events to third-party SIEM platforms.
  - Policy-based access control: rules based on user roles, JWT claims, tenants, agent types.
- Deployment: cloud SaaS, no-code/low-code.
- Target: MCP server developers, AI agent builders, enterprise security teams; cites survey 88% using/planning AI agents.
- Quote: "AI agents are breaking traditional identity systems... they can't be managed like human users or service accounts."
