January 2026: Nightfall announced "AI Browser Security," targeting data exfiltration from
employees using ChatGPT, Claude, Gemini — described as "the fastest-growing data loss vector"

Operates without proxies or SSL inspection; real-time interception across three layers:

- Endpoint: cloud sync tools, desktop AI apps, Git operations, USB transfers, printing.

AI-native engine: ML + LLMs + computer vision/OCR; detects credentials, PII, payment data,
health info, and business assets (source code, financial projections).

CEO Rohan Sathe: "Legacy data loss prevention was never designed to see or understand those
