## Key extracted text

What it is: Enterprise AI governance platform that standardizes connections between AI clients
and enterprise tools; lets AI agents securely access business systems while maintaining
compliance and control.

MCP gateway features:
- "Verified MCP server library" + custom MCP deployment across cloud, desktop, self-hosted.
- Centralized configuration for AI clients/agents connecting to enterprise systems.

Access & authz:
- Role/intent-based toolkits called "Profiles" (predefined tool collections).
- "Attribute-based authorization" with identity-aware, context-aware enforcement.
- Credential management: managed or bring-your-own.

Visibility/compliance:
- Complete audit trails of all tool access/activity.
- "Shadow AI discovery and control" (claims avg 225 unauthorized AI deployments per org).
- SIEM, EDR, and MDM integration.

Deployment: hybrid — cloud-hosted, VPC, on-premises, desktop (local tool access).

Target: enterprises needing secure AI agent deployment with strict governance/compliance
(finance, manufacturing, retail, healthcare, IT ops). SOC2 certified; GDPR/CCPA compliant.
