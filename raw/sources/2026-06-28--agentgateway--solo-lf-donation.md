Donation date: 2025-08-25. Recipient: Linux Foundation.

Why: Solo.io created agentgateway to fill a gap — traditional API gateways predate AI
agents and lack agent-protocol support. Donated to keep it "vendor-agnostic and
community-driven" under neutral governance.

Solo.io's role: created agentgateway applying lessons from building Istio's zero-trust
tunnel (ztunnel) over three years. Purpose-built in Rust for AI agent communication.

Relationship to kgateway: kgateway is a SEPARATE CNCF-hosted project built on Envoy,
distinct from agentgateway (purpose-built in Rust for AI protocols). [Note: agentgateway
can serve as an AI-native data plane that kgateway's control plane can manage in k8s.]

Early contributors / community participants (since July 2025): AWS, Microsoft, Red Hat,
IBM, Zayo, Shell, Cisco, Huawei.

Project creation: not given precisely; community meetings began July 2025, preceding the
Aug 25 donation. (Other sources say created ~March 2025.)

Quote: "The future of software is agentic and that changes everything about how systems
connect and communicate" — Idit Levine, CEO of Solo.io.
