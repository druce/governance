# Socket Series B (vendor blog extract)

- Series B: $40M, announced October 22, 2024.
- Lead: Abstract Ventures; participation from Elad Gil, Andreessen Horowitz (a16z), angels incl. Bret Taylor.
- Total funding to date: $65M.
- Founder/CEO: Feross Aboukhadijeh (open-source dev; created WebTorrent, StandardJS).

- Scale claims: blocks 100+ supply chain attacks/week; protects 7,500+ organizations and 300,000+ GitHub repos.
- Deployment: GitHub App, CLI, pull-request alerts/comments.
