# Microsoft Sentinel — the security platform for the agentic era (Microsoft Security Blog)

URL: https://www.microsoft.com/en-us/security/blog/2025/09/30/empowering-defenders-in-the-era-of-agentic-ai-with-microsoft-sentinel/
Fetched: 2026-06-28
Good for: Confirming Sentinel is a Microsoft-owned cloud-native SIEM and its 2025-26 pivot to an agentic AI platform (Security Copilot, data lake, graph, MCP server).

## Key extracted facts
- Microsoft Sentinel is a cloud-native SIEM, owned/developed by Microsoft as part of its security portfolio.
- Now positioned as a unified security data lake + agentic AI platform; ingests signals, builds context via vectorized data and graph relationships.
- Security Copilot integration: agents reason across environments using Sentinel's graph context, correlate alerts, automate response.
- Sentinel data lake: launched July 2025; GA announced Sep 30 2025.
- MCP (Model Context Protocol) server: public preview — semantic access + agentic orchestration; agents built in Security Copilot or VS Code/GitHub Copilot can act on security data.
- No-code agent builder in Security Copilot portal (natural-language agent creation).
- Dated: Mar 2025 Security Copilot agents launched; Jul 2025 data lake; Sep 30 2025 agentic platform announcement.
- March 2026 updates (per other MS sources): Sentinel playbook generator (NL -> Python playbook), Copilot data connector preview, agent identities asset connector.
