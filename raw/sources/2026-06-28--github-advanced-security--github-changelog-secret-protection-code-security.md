## Key extracted text

- Announcement date: March 4, 2025. GitHub introduced two standalone security products replacing the bundled GitHub Advanced Security (GHAS).
- GitHub Secret Protection — $19/month per active committer. Push protection to prevent secret leaks before they happen; AI-powered detection with low false positives; secret scanning alerts and custom pattern matching; security overview dashboard; enterprise-scale governance.
- GitHub Code Security — $30/month per active committer. Copilot Autofix for vulnerability remediation; security campaigns; Dependabot features for dependency-based vulnerabilities; security overview; third-party tool integration.
- Availability: starting April 1, 2025, GitHub Team customers can purchase these products through metered (consumption-based, pay-as-you-go) billing — expanding access beyond Enterprise tiers.
- Both available as standalone products; can be adopted independently.
