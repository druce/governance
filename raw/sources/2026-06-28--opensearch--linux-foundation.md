Good for: Ownership/governance of OpenSearch — AWS transferred it to the Linux Foundation (OpenSearch Software Foundation), Sept 2024. Confirms vector-database use.

Launch date: September 16, 2024. AWS transitioned OpenSearch to the Linux Foundation,
establishing the OpenSearch Software Foundation as a "vendor-neutral community for
search, analytics, observability, and vector database software."

OpenSearch enables developers to "build search, analytics, observability, and vector
database applications." Established 2021 under AWS (as a fork of Elasticsearch/Kibana
7.10 after Elastic's license change); 700M+ downloads, 200+ maintainers.

Founding members: Premier — AWS, SAP, Uber. General — Aiven, Aryn, Atlassian,
Canonical, DigitalOcean, Eliatra, Graylog, NetApp Instaclustr, Portal26.

Note: OpenSearch is Apache-2.0 licensed open source. AWS still offers the managed
"Amazon OpenSearch Service," but the project itself is now Linux-Foundation-governed.
