## Key extracted facts
- Microsoft Defender XDR is a Microsoft product (public, NASDAQ: MSFT). Unifies Defender for Endpoint (EDR), Defender for Office 365, Defender for Identity, Defender for Cloud Apps into one XDR.
- Defender for Endpoint: cloud-native EDR, mobile threat protection, advanced hunting.
- Security Copilot in Defender: Copilot Chat in Defender went GA April 2026 — conversational, context-aware SOC interface.
- Agents: Phishing Triage Agent (GA), Dynamic Threat Detection Agent (continuous backend correlation across Defender + Sentinel), Security Alert Triage Agent (extends agentic triage to identity/cloud alerts, preview from April 2026).
- Agentic endpoint security (part of A365): discovers/governs/blocks AI agents running locally on endpoints; Defender auto-discovers local AI agents on onboarded Windows devices, surfaced in AI agent inventory + exposure map + advanced hunting.
- Named a Leader in 2026 Gartner Magic Quadrant for Endpoint Protection.
