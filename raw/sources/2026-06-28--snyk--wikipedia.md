# Snyk (Wikipedia extract)

- Founding Year: 2015 (founded out of Tel Aviv and London)
- Founders: Guy Podjarny, Assaf Hefetz, Danny Grander
- Headquarters: Boston, Massachusetts, U.S.
- Ownership: Private (independent; VC-backed)
- Total funding raised: ~$745M+ per the Wikipedia text (note: this figure is dated; later rounds push total higher — see getlatka/Crunchbase which cite ~$1.6B total)
- Recent funding noted: January 2023 ServiceNow strategic investment $25M
- Valuation: $8.5B (September 2021, Series F round per Wikipedia); subsequent Series G (Dec 2022) reportedly at ~$7.4B
- Product: "developer-oriented cybersecurity company, specializing in securing custom-developed code, open-source dependencies and cloud infrastructure." Finds vulnerabilities in code before deployment.
- Acquisitions: DeepCode (Sep 2020), Manifold (Jan 2021), FossID (May 2021; reacquired by founders Sep 2022), CloudSkiff (Oct 2021), Fugue (Feb 2022), Probely (Nov 2024, DAST), Invariant Labs (Jun 2025, AI security research).
