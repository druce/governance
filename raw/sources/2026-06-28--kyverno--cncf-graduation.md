Kyverno is a Kubernetes-native policy engine for defining, managing, and enforcing
policy-as-code across cloud-native environments.

Originally created by Nirmata; contributed to CNCF in 2020. Timeline:
- Accepted to CNCF: 2020-11-10
- Incubating: 2022-07-13
- Graduated: announced 2026-03-24 (moved to graduated maturity 2026-03-16),
  KubeCon EU Amsterdam. Joins ~35 graduated projects (Kubernetes, Prometheus, Envoy, Helm).

Quote (Chris Aniszczyk, CNCF CTO): "Kyverno's graduation highlights how important
policy-as-code has become for organizations running cloud native in production at scale."

Grew from 574 GitHub stars to 9,000+; adopters include Bloomberg, Coinbase,
Deutsche Telekom, LinkedIn, Spotify, Vodafone, Wayfair. Apache-2.0 OSS.
Commercial support / backing from Nirmata.
