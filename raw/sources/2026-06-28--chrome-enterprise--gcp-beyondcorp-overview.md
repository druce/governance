Note: Google Cloud documentation establishing that Chrome Enterprise Premium is the renamed/successor branding of BeyondCorp Enterprise (Google's Zero Trust access product), and the DLP integration model.

## Key extracted text (from Google Workspace/Cloud docs + search synthesis)

- Chrome Enterprise Premium is Google Cloud's secure enterprise browsing solution, delivering centralized management, threat and data protection, and Zero Trust access controls to web applications. It builds on Chrome's secure foundation.
- Lineage: Chrome Enterprise Premium is the rebrand/evolution of BeyondCorp Enterprise — Google's context-aware access / Zero Trust product. Context-aware access enforces secure access to apps and resources based on user identity, device security posture, and geographic location.
- DLP: with Chrome Enterprise Premium DLP, admins can scan up to 10 MB of text content in a file to detect data that is opened, uploaded, downloaded, pasted, or transferred; enforce data protection policies plus malware/phishing protections across desktop and mobile.
- Tiering: Chrome Enterprise Core is available at $0 (management/console layer); Premium is the paid security upgrade.
- Integrations: BeyondCorp/Zero Trust access; Google Security Operations (Google SecOps / Chronicle) for telemetry; works with third-party SASE (e.g., Cisco Secure Access integration guide exists).
