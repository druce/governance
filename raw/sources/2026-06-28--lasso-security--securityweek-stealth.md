## Key text (trimmed)

Lasso Security, Tel Aviv, Israel; founded 2023; emerged from stealth November 2023.
CEO Elad Schulman. $6M seed led by Entrée Capital with Samsung Next participation.

Provides end-to-end generative AI security: "tackle the cyber threats faced by
generative AI and large language models (LLMs) and prevent data exposure, and security
and compliance risks." Prevents data poisoning, model theft, malicious code generation,
prompt injection, supply-chain attacks. Monitors sensitive data disclosures; identifies
LLM tools used by employees with real-time safety guidance; oversight across "every LLM
touchpoint."
