## Key extracted text

Chrome Enterprise Premium is an advanced security solution built on top of Chrome Enterprise Core's management capabilities. Positioned as "the future of endpoint security." Price: $6 per user monthly (vendor list price, as of fetch date — marketing).

Security features:
- Data Loss Prevention (DLP): prevent accidental and intentional exfiltration of company-sensitive data through granular policies, including controls for unsanctioned AI tools / shadow AI.
- Threat Protection: real-time malware and phishing defenses (vs standard Safe Browsing in free tier), malware deep scanning for unknown/high-risk files, advanced threat detection.
- Context-Aware Access Control: restricts access to SaaS apps, Google Cloud, and private web apps based on user identity, geographic location, and device security status — aligned with Zero Trust (this is the former BeyondCorp Enterprise capability).
- Security Insights & Response: visibility into risky users, sensitive data transfers, and shadow AI activity.

Target: mid-to-large enterprises with hybrid/remote workforces needing endpoint security beyond basic browser management.
Deployment: cloud-based management across OSes; third-party integrations to existing security tools.
Customer reference (marketing): Snap reported "50% reduction in content transfers" after DLP restrictions for generative AI platforms.
