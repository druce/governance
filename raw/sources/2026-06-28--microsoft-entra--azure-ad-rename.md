Good for: Microsoft Entra ID identity, rename history, ownership.

Key facts:
- Microsoft Entra ID is the new name (announced July 2023) for Azure Active Directory (Azure AD). All features/capabilities unchanged; rename only.
- "Microsoft Entra" is the product family of identity + network access solutions; Entra ID is the cloud IdP within it (SSO, MFA, Conditional Access, lifecycle, identity protection). Other Entra products: Entra ID Governance (IGA), Entra Permissions Management (CIEM), Entra Verified ID, Entra Internet/Private Access (SSE), Entra Agent ID (agent identity).
- Owned by Microsoft (Nasdaq: MSFT). Bundled with Microsoft 365 / Azure; the default IdP for orgs already on Microsoft.
