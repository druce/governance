# Lasso Security — Open-Source MCP Security Gateway (vendor primary)

- URL: https://www.lasso.security/resources/lasso-releases-first-open-source-security-gateway-for-mcp
- Fetched: 2026-06-28
- Good for: agent/MCP runtime security positioning, deployment (open-source proxy).

## Key text (trimmed)

Announced 2025-04-15. Open source, available on GitHub, Smithery, and MCP Servers as a
proxy/orchestrator. Described as "the first security-centric solution for Model Context
Protocol (MCP) specifically designed with agentic workflows development in mind."

Acts as an intermediary for MCP interactions, embedding security controls into agentic
workflows. Capabilities:
- Configurable guardrails — filter incoming requests and outgoing responses to prevent
  sensitive data leakage before info reaches AI agents.
- Usage analytics — visibility into tool interactions.
- Centralized monitoring — dashboard for all MCPs in a single interface.
- Risk mitigation — prompt injection, command injection/RCE, credential exposure,
  logging gaps in agentic systems.

CTO quote: lets teams "scale GenAI agent development without compromising on safety,
confidentiality, and transparency," while staying compatible with existing agent
implementations.
