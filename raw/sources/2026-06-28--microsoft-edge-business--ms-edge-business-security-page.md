## Key extracted text

"Edge for Business offers enterprise-grade security, integrating benefits from Microsoft 365 at no extra cost." Secure browser to protect data across managed devices, BYOD, contractor devices, and mobile.

Security features:
- Data Protection: Microsoft Purview sensitivity labels with usage-rights restrictions, DLP policies, watermarking, copy/paste boundary controls.
- Authentication: Zero Trust via strong authentication; native Microsoft Entra Conditional Access (role-based access controls, governance).
- Threat Prevention: built-in Microsoft Defender for Endpoint integration and Defender SmartScreen (phishing/malware blocking).
- AI Governance: content-aware controls for blocking risky prompts on unsanctioned GenAI apps.
- Mobile: enterprise protection extended to iOS and Android via Intune management.
- Device flexibility: managed Windows PCs, personal devices, contractor machines without device management requirement.

Native integrations: "Entra, Purview, Intune, and Microsoft Defender for Endpoint" — no third-party extensions required.
Licensing: included with Microsoft 365 subscriptions at no additional cost (an enterprise licensing matrix details specific entitlements; some advanced controls require specific M365/Purview tiers).
Target: enterprises of all sizes securing data across hybrid work without added complexity.
