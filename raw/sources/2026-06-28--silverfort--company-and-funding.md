# Silverfort — company profile + funding

- URLs: https://www.silverfort.com/company/ ; https://techcrunch.com/2024/01/23/silverfort-now-valued-at-1b-after-raising-116m-for-its-holistic-approach-to-identity-security/ ; https://en.wikipedia.org/wiki/Silverfort
- Fetched: 2026-06-28
- Good for: product, founders, funding/valuation, NHI + runtime identity focus.

## Key extracted facts
- Product: unified identity security platform that protects human and non-human identities across cloud and on-prem. Known for agentless/proxyless approach that extends MFA and conditional access to systems that couldn't previously support it (legacy apps, service accounts, command-line, OT). Runtime Access Protection (RAP); dedicated Non-Human Identity (NHI) security (discover/monitor/protect machine identities).
- Founded: 2016, by Hed Kovetz (CEO), Yaron Kassner (CTO), Matan Fattal.
- HQ: Tel Aviv, Israel + Boston/US (sources vary: Tel Aviv, Dallas/Plano TX listed). Israeli-origin, US dual presence.
- Funding: ~$222M total. Series D $116M Jan 2024 led by Brighton Park Capital, valuation ~$1B (unicorn). Earlier: $11.5M A, $30M B, $65M C.
- 2025: Howard Greenfield brought on as President/CRO.
- Differentiator (marketing): "the only platform that truly goes everywhere" — single layer for identity protection, context, visibility.
