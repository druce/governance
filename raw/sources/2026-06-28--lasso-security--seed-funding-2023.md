## Key text (trimmed)

Announcement date: 2023-11-20. Lasso Security emerged from stealth with a **$6 million
seed round led by Entrée Capital, with participation from Samsung Next**.

Founded 2023; HQ **Tel Aviv, Israel**. CEO/co-founder **Elad Schulman**. (Press lists
co-founders Lior Ziv, Ophir Dror, Yuval Abadi.)

Platform provides "observability, governance, and real-time defense across the entire AI
ecosystem." Capabilities:
- Shadow AI Discovery — identifies unauthorized LLM usage.
- Data-flow monitoring & observability — tracks LLM interactions.
- Real-time detection & alerting.
- Threat protection — model theft, prompt injection, data poisoning, supply-chain
  attacks, malicious code generation.

Quote: "Every data point, command, and prompt generated by LLMs will be under the
vigilant oversight of our sophisticated security platform" — Elad Schulman, CEO.

Targets enterprises integrating LLMs into production while safeguarding sensitive data
and maintaining compliance.
