## Key extracted text

- Obot is "an open-source platform that provides everything an organization needs to implement MCP technologies." Tagline: "Complete MCP Platform — Hosting, Registry, Gateway, and Chat Client."

Core platform features:
- **MCP Hosting**: Run and manage MCP servers with Docker or Kubernetes; supports Node.js, Python, and container-based servers; single-user and multi-user scenarios.
- **MCP Registry**: "central place to list and discover MCP servers" with a curated catalog, conforms to MCP registry specifications.
- **MCP Gateway**: "a single entry point for accessing MCP servers" with access rules, logging, request filtering.
- **Obot Chat**: client supporting multiple model providers (OpenAI, Anthropic), built-in RAG, project-wide memory, scheduled task automation.

Security & access control:
- "OAuth 2.1 and token handling for authentication," access controls by user/group, comprehensive audit logging, "encryption at rest and in transit."

Deployment: self-hosted on your own infrastructure with Docker or Kubernetes.
License: MIT.
