## Key extracted text

- Founders: Alex Atallah and Louis Vichy.
- Product: OpenRouter is a unified API platform / "control plane" for managing LLM inference. Single API access to hundreds of LLMs across providers; handles failover, load balancing, routing.
- Capabilities: model selection based on data privacy policies; automatic failover; output control; external source integration.
- Funding: round led by Andreessen Horowitz (a16z). Announcement date June 26, 2025. (Amount/valuation not disclosed in this post.)
- Adoption: "the OpenRouter API is used by more than 1 million developers, and that users are now spending 10x more on inference than they were just 7 months ago."
- Thesis: addresses LLM fragmentation where "models flicker on and off, pricing changes overnight, and providers have different APIs."
