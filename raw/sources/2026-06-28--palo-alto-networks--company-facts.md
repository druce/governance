## Facts
- Founded 2005 (Nir Zuk). HQ Santa Clara, CA. Public: NASDAQ: PANW.
- CEO/Chairman Nikesh Arora (since June 2018).
- FY2025 revenue ~$9.22B.
- Three platforms: Strata (network security/SASE), Cortex (SecOps/XDR/XSIAM), Prisma Cloud (CNAPP) + Prisma AIRS (AI security). Plus new Identity Security pillar via CyberArk.

## AI-security roll-up (verified dates this research)
- Talon Cyber Security -> Prisma Access Browser. Closed 2023-12-28.
- Protect AI -> Prisma AIRS. Closed 2025-07-22.
- CyberArk (identity, incl. Conjur secrets) -> new Identity Security pillar. Announced 2025-07-30; closed 2026-02-11; ~$25B.
- Portkey -> AI Gateway component of Prisma AIRS. Announced 2026-04-30; closed 2026-05-29; terms undisclosed.
