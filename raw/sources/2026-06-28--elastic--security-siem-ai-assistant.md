## Key extracted facts
- Elastic N.V. is a public company, NYSE: ESTC. HQ effectively distributed; incorporated in Netherlands, US ops Mountain View/San Francisco. Founded 2012 (Elasticsearch created 2010 by Shay Banon).
- Elastic Security is an agentic security operations platform unifying SIEM, XDR, and automation, built on the Elasticsearch (ELK = Elasticsearch, Logstash, Kibana) stack.
- Elastic AI Assistant built on Elasticsearch Relevance Engine (ESRE); generative-AI guidance for alert triage, incident response. AI assistant exposes a reasoning trace (prompts, tool calls, responses). Customers can connect multiple LLMs and select per workflow.
- Elastic AI SOC Engine (EASE) announced to fight alert fatigue; works alongside Splunk, Microsoft Sentinel.
- Named a Leader in IDC MarketScape Worldwide SIEM 2026 (June 2026).
- Open-source roots; relicensed (SSPL/Elastic License) 2021; re-added AGPL option 2024. OpenSearch is the AWS fork.
