Good for: Claude Compliance API capabilities + archiving/eDiscovery/SIEM integration partners (comms-surveillance relevance for hedge funds).

What it provides (by tier):
- Claude Enterprise: API exposes "conversation content including chats, uploaded files, and projects" plus "activity feed events covering user logins, admin actions, and configuration changes."
- Claude Platform (API customers): only "activity feed events" (admin/system + resource events); conversation content excluded.

Availability: "available to Claude Enterprise plans and Claude Platform customers."

Integration partners (60+; expanded from 28 announced 2026-05). Comms-surveillance / archiving / eDiscovery relevant:
- Proofpoint - DLP on prompts/responses; captures activity "for eDiscovery, supervision, and compliance"
- Smarsh - "Monitor and retain Claude Enterprise interactions"; structured access to conversations + audit support (Smarsh Capture for Claude Enterprise, announced 2026-05-21)
- Theta Lake - "Investigation, Analytics, and Regulatory Compliance capabilities"
- RelativityOne - collects conversation data, normalizes into "Relativity Short Message Format (RSMF)" for review
Other categories: SIEM/SOC (CrowdStrike, Elastic, Sumo Logic, ReliaQuest); DLP (Forcepoint, Nightfall AI, Trellix, Zscaler); Identity (Okta, SailPoint, Saviynt, Linx Security); AI security (Akto, Harmonic Security, SentinelOne).

Context: Compliance API launched 2025-08-20; ~30 typed audit events; 180-day retention window for audit/activity events. Anthropic announced 28 security/compliance integrations 2026-05-25 (Help Net Security), since grown to 60+.
