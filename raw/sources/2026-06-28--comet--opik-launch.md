## Key extracted facts
- Comet founded 2017; HQ New York City; distributed team across 14 countries. Raised ~$70M (Series B led by OpenView; earlier seed rounds led by Trilogy Equity Partners). 150+ enterprise customers incl. Netflix, Uber, Cisco, Etsy, Zappos.
- Comet's core platform: ML experiment management, model management, production monitoring (the legacy MLOps product, predates LLMs).
- Opik launched 2024-09-17 as a fully open-source LLM evaluation/observability platform (Apache-2.0; code on GitHub, ~19k+ stars, integrates 40+ frameworks/providers).
- Opik features: record/sort/search each step an LLM app takes (tracing), built-in eval metrics + LLM judges (hallucination, factuality), model unit testing, scoring, production dashboards.
- Deployment: self-host via OSS, or Comet-managed/enterprise version with collaboration, user management, security.
