# Infisical — open-source secrets management funding/profile

- URLs: https://www.ycombinator.com/companies/infisical ; https://infisical.com/ ; https://github.com/Infisical/infisical ; https://www.crunchbase.com/organization/infisical

- Good for: funding, founders, HQ, open-source model, agents/NHI focus.

## Key extracted facts
- Product: open-source platform for secrets, certificates, and privileged access management. Stores/injects secrets across dev workflows, CI/CD, cloud infra; secrets versioning, point-in-time recovery, audit logging, automatic rotation. Self-hosted, end-to-end encrypted option (vs cloud-only). 40M+ downloads claimed; large GitHub presence.
- Founded: 2022, by Vlad Matsiiako (CEO), Tony Dang, Maidul Islam. HQ: San Francisco. Y Combinator W23.
- Funding: $16M Series A led by Elad Gil (reported ~June 2025), with Y Combinator, Gradient, Dynamic Fund + angels (Datadog CEO Olivier Pomel, Samsara CEO Sanjit Biswas). Independent, VC-backed.
- Positioning: open-source / self-hostable challenger to Doppler (SaaS) and HashiCorp Vault. Increasingly pitching secrets + identity for AI agents.
