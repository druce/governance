Key extracted text:
- Founded 2008 in Fulton, Maryland by Apache Maven creators Brian Fox and Jason van Zyl.
- Born when open-source component use exploded (2008) to guard the Java Central Repository (Maven
  Central) and automate management of third-party libraries.
- First product: Nexus Repository Manager (private artifact repository for enterprises).
- Stewards The Central Repository (Maven Central), the largest repository of Java OSS components.
- 2012: added Nexus Lifecycle (policy enforcement across build/test/deploy) — shift from tooling to platform.
- Early funding: ~$11M Series A led by Accel Partners and Bay Partners.
- Additional offices: London, Sydney, Hyderabad.
