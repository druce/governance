# Hydden — $4.4M seed round

- URL: https://www.securityweek.com/hydden-raises-4-4m-in-seed-funding-for-identity-security-platform/
- Also: https://hydden.com/news/hydden-secures-4-4m-seed-round/ ; https://www.darkreading.com/identity-access-management-security/startup-finds-hydden-identities-it-environment

- Good for: funding, founder, product positioning vs IGA/PAM/ITDR.

## Key extracted facts
- Product: identity security platform / "identity system of record" providing visibility into every identity, account, and privilege across hybrid infrastructure. Creates a single data layer across IAM, IGA, PAM, and ITDR tools without disrupting existing infrastructure.
- CEO / co-founder: Jai Dargan. Founded ~2022 (per secondary sources; not stated in article). HQ: New York (per secondary sources).
- Seed: $4.4M, announced 2024-09-17. Lead: Access Venture Partners; with Lockstep (Rinki Sethi & Lucas Moody CISO fund), Service Provider Capital, cyber angels (Andy Grolnick, Paul Trulove).
- Positioning quote: "Siloed IAM solutions, built for a different era, can't keep pace with the complexities of modern, hybrid infrastructures."
- Note: visibility/correlation layer over existing identity stack, not a full IGA suite. Small seed-stage company.
